# EvoDeck roadmap

## Milestone 1 – Stable Core ✅

Flask application shell, SQLite card catalog, set and card browsing, card-detail data, restored card API, repository hygiene, and the initial collection engine.

## Milestone 2 – Collection Experience

Improve collection browsing and editing, add clearer summary/progress views, and make collection-item management more direct.

## Milestone 3 – Variants

Connect catalog variant definitions to collection entries and support variant-specific completion tracking.

## Milestone 4 – Storage

Add structured storage locations, binders, and placement tracking beyond the current free-text location field.

## Milestone 5 – Pricing

Use the pricing service to import market prices, track collection value, and compare acquisition cost with current value.

## Milestone 6 – Multi-user

Add authentication and user management, replacing the current default-user assumption with authenticated ownership boundaries.

## Milestone 7 – Marketplace & Trading

Support trade lists, marketplace workflows, offers, and collection-sharing features.
